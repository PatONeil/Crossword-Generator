This is a full function crossword generator for 15x15 puzzles.   It allows custom layouts,  partial entry of word for themee and seeding puzzle, a solver for filling in the remainder of the puzzle, 
the ability to review clues provided or enter your own clues and export in either json or puz format.   It contains a dictionary of over a 100,000 words.

The application is written as a browser based PWA application written entirely in Javascript.

You can view a demo at https://pjoneil.net/CW_Generator
